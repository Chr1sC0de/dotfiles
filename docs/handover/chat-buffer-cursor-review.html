Codex Chat Buffer Cursor

This handover explains the cursor-placement change for the Codex Chat Buffers floating panel and gives reviewers a direct path through the implementation, tests, and acceptance criteria.

What Changed

Newly opened chat-buffer panels now move the cursor to the first selectable chat row after rendering. The row is discovered from the existing state.codex_chat_line_to_buf mapping.

Refreshes and focusing an already-open panel still render in place without moving the user's cursor.

Acceptance Criteria
